# Spotify Visualizer PRO

Una aplicación web profesional para visualizar tu música de Spotify en tiempo real con efectos increíbles.

## Características

- Visualización en tiempo real con múltiples efectos
- Top 10 canciones y artistas
- Historial de reproducción reciente
- Estadísticas detalladas de tu música
- Interfaz moderna con diseño glassmorphism
- Actualización automática de tokens
